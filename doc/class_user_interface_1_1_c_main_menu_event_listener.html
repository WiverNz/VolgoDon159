void UserInterface::CMainMenuEventListener::OnUIEvent ( IUIElement *  pSender,
const SUIEventDesc &  event,
const SUIArguments &  args 
)
overridevirtual

This is called, if an event (fscommand) is triggered from flash. This function is only called if the event is found in the UI Element XML file. The OnUIEvent callback function will only handle events, which are defined in the UI Element XML file. There is also a OnUIEventEx callback function, which will trigger for other fscommands coming from your UI element.

Parameters
pSender{IUIElement*} the UI element that sent the event
event{SUIEventDesc} the UI event that occurred
args{SUIArguments} the arguments that are tied to the event
