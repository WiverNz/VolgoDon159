Detailed Description

template<typename T>
class CryGTestDetails::CLeftHandSideExpression< T >

Represents everything on the left hand side of a comparison operator. The evaluation of a comparison can be either a CBinaryExpression or folded back to a CLeftHandSideExpression.

For convenience sometimes we want to use '&&' or '||' in the expression. The expression is collapsed to the result of '&&' or '||'. The output misses the nicer actual values, but then the user is responsible for using a simpler expression instead of '&&' or '||'.
